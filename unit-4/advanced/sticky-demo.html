Modern Layout Notes

CSS Grid is useful when the page needs control in two directions.

Container queries are helpful because components respond to their own space instead of the viewport.

Subgrid lets nested content align to tracks created by a parent grid.

Sticky positioning keeps supporting content nearby without removing it from the normal document flow.

These layout tools work best when each one solves a specific layout problem.

Using them together can make a page more flexible, responsive, and maintainable.

Key Ideas

  • Subgrid aligns internal card content.
  • Container queries make components context-aware.
  • Sticky modules keep helpful content visible.
