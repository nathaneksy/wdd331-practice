Filter vs Backdrop-Filter Demo

Scroll this page with Chrome DevTools Paint Flashing.

Demo Instructions

  1. Open Chrome DevTools.
  2. Press Ctrl + Shift + P and search Show paint flashing.
  3. Scroll the page and watch the sticky header repaint.
  4. In the CSS file, switch the header class styles to compare performance.

Try changing the header class in HTML:

backdrop-blur cheap-solid heavy-backdrop-blur

No Filter

This is the baseline. No filter effect is being applied.

filter: blur(4px)

This blurs the image itself. Small, static filters are usually fine.

filter: blur(14px)

Larger blur values increase GPU work because more neighboring pixels are sampled.

Backdrop-Filter Panel

This panel uses backdrop-filter, which filters what is behind the element.

Try scrolling while watching paint flashing in DevTools.

Scrollable Image Grid

The sticky header sits over this content. That makes backdrop-filter more expensive during scroll.
